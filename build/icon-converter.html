ClipForge Icon Converter

Convert the SVG icon to PNG at various resolutions for Electron app icons.

Preview (512x512)

Next Steps:

  1. Download the 1024x1024 PNG (for best quality)
  2. Install electron-icon-maker: npm install --save-dev electron-icon-maker
  3. Generate icons: npx electron-icon-maker --input=./build/icon-1024.png --output=./build
  4. Or manually save the 512x512 as icon.png for a quick test
